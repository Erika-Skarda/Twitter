## Como rodar a aplicação :arrow_forward:

No terminal, clone o projeto: 

```
git clone https://github.com/Erika-Skarda/Twitter
```
Entre no projeto e instale as dependências através do comando:
```
npm install
```
Por último, suba a aplicação:
```
npm start
```
